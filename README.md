# neighborhood-map
A node.js based application along with third-party APIs as Google Maps API and Yelp Fusion.

Prerequisites
-------------
1. Node.js

How to run the app?
-------------------

Clone the project: `git clone https://github.com/csukeyosi/neighborhood-map.git`

Go to the project's root directory and install the dependencies: `npm install`

Run the server: `npm start`

Check out the app: http://localhost:5000/
